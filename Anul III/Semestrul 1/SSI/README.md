# Sisteme de Securitate Informatice
### Grading

 - ```30p``` -> laborator (```24p momentan```)
 - ```10p``` -> teme (moodle)
 - ```60p``` -> examen

### Conditii de promovare:

```
>= 50% in total (inseamna min 10p din 60p la examenul final daca s-a obtinut punctajul maxim pe parcursul semestrului)
```
